# VPN Traffic Detector

Проект посвящён классификации VPN-трафика на основе метаданных сетевых потоков с использованием методов машинного обучения.

## Кратко о проекте

Реализован полный цикл работы с моделями:
- обучение моделей (разные архитектуры)
- настройка предобработки данных
- сравнение моделей
- анализ качества и устойчивости

## Основные возможности

- Единый pipeline: Preprocessing → Scaling → (PCA) → Model
- Поддержка нескольких моделей:
  - Random Forest
  - Gradient Boosting
  - XGBoost
  - Logistic Regression
  - Stacking
- Набор профилей предобработки данных
- Автоматическая оценка (train/test split 70/30)
- Сохранение результатов экспериментов
