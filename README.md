# 📘 README (упрощённая версия для преподавателя)

## VPN Traffic Detector

Проект посвящён классификации VPN-трафика на основе метаданных сетевых потоков с использованием методов машинного обучения.

---

## Кратко о проекте

Реализован полный цикл работы с моделями:

* обучение моделей (разные архитектуры)
* настройка предобработки данных
* сравнение моделей
* анализ качества и устойчивости

---

## Основные возможности

* Единый pipeline:
  **Preprocessing → Scaling → (PCA) → Model**

* Поддержка моделей:

  * Random Forest
  * Gradient Boosting
  * XGBoost
  * Logistic Regression
  * Stacking

* Профили предобработки данных

* Автоматическая оценка (train/test split 70/30)

* Сохранение результатов экспериментов

---

## Веб-интерфейс

### Обучение моделей

```bash
streamlit run src/train.py
```

Позволяет:

* выбрать модель
* выбрать профиль предобработки
* включить/выключить PCA
* обучить модель и сохранить результат

---

### Анализ результатов

```bash
streamlit run src/analyze.py
```

Позволяет:

* сравнивать модели
* анализировать метрики
* смотреть важность признаков
* исследовать устойчивость моделей

---

## Данные

Ожидается CSV-файл с колонкой:

* `traffic_type` — метка (VPN / NonVPN)

---

## Структура проекта

```
src/
  core.py        — pipeline и препроцессинг
  config.py      — модели и профили
  train.py       — обучение (web)
  analyze.py     — анализ (web)

data/processed/  — данные
models/          — сохранённые модели
results/         — отчёты экспериментов
```
